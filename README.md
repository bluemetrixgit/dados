# App Dados
**Sobre o app**<br>
Um dashboard com três abas que retorna as informações gerais da planilha de controle, podendo fazer cinsultas de acordo com as datas que o usuário desejar, como:
**Primeira aba (Dados gerais de Bluemetrix)**
+ PL
+ PL dos assessores
+ Distribuiçao de conta entre os assessores
+ Evolução dos PL's
+ Distribuição dos patrimônios
**Segunda aba (Taxa de gestão)**
  + Calcula o valor da taxa de gestão 
**Terceira aba (Enquadramento de contas)**
+ Identifica os clientes que não estão de acordo com a estratégia.
